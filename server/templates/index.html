FastAPI Demo Project

A basic project to demonstrate FastAPI with Load Balancing

About

This is a demo project built using FastAPI to showcase high performance and scalability. It includes a basic load balancer setup for routing requests across multiple backend instances.

Architecture

The project uses the following components:

  • FastAPI app instances running
  • A load balancer in front of the app
